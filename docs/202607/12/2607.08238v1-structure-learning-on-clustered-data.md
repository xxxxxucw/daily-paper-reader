## 摘要
最近算法进展使得有向无环图（DAG）结构学习在因果发现中具有可扩展性。然而，现有技术假设完全同质的总体，因而无法应用于聚类数据——其中聚类特定的变异（例如患者特异性效应）很常见。我们通过引入一种新方法来解决这一问题，该方法在考虑局部聚类级效应的同时估计全局结构。关键思路是将经典混合模型的固定效应和随机效应框架扩展到结构学习场景。为此，我们提出一种可微的图耦合机制，确保固定效应和随机效应图的并集保持无环。计算上，我们提供了一种可证明收敛的一阶方法，并跨聚类利用高效的批量更新。统计上，我们建立了模型的可识别性，并表明我们的方法渐近地恢复真实结构。在真实和合成数据的实验中，我们的方法检测到了替代估计器遗漏的依赖关系，这突显了其在聚类环境中进行结构学习的价值。

## Abstract
Recent algorithmic advances have made directed acyclic graph (DAG) structure learning scalable for causal discovery. Yet, the currently available techniques assume a completely homogeneous population, precluding their application to clustered data where cluster-specific variations (e.g., patient-specific effects) are common. We address this issue by introducing a new approach that estimates a global structure while accounting for local cluster-level effects. The key idea is to extend the fixed- and random-effects framework of classical mixed models to the structure learning setting. Towards this end, we present a differentiable graph coupling mechanism that guarantees the union of the fixed- and random-effects graphs remains acyclic. Computationally, we provide a provably convergent first-order method and leverage efficient batched updates across clusters. Statistically, we establish identifiability of the model and show that our approach recovers the true structure asymptotically. In experiments on real and synthetic data, our proposal detects dependencies missed by alternative estimators, underscoring its value for structure learning in clustered settings.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：现有的有向无环图（DAG）结构学习方法（如NOTEARS、DAGMA等）均假设总体完全同质，无法处理聚类数据中存在的**集群特异性变化**（如患者个体效应）。在医学、生物学等领域的聚类数据（如患者重复测量、肿瘤免疫微环境）中，这种假设不成立。
- **研究背景**：传统的混合效应模型能够处理固定效应与随机效应，但此前未与可微结构学习框架结合。可微无环性约束（如Zheng et al., 2018）将组合优化转化为连续优化，为融合混合效应提供了可能。
- **整体含义**：本文首次提出**混合DAG**（Mixed DAG）框架，将线性混合模型中的固定效应和随机效应框架扩展到结构学习场景，同时保证固定图与随机效应图的并集无环，从而在聚类数据中同时学习全局因果结构与局部变异。

## 2. 论文提出的方法论：核心思想、关键技术细节、算法流程

- **核心思想**：聚类数据中每个集群的加权邻接矩阵 \( \mathbf{W}^{(i)} \) 分解为**全局固定效应矩阵 \(\mathbf{B}\)** 与**集群特定随机效应矩阵 \(\mathbf{U}^{(i)}\)**，即 \(\mathbf{W}^{(i)} = \mathbf{B} + \mathbf{U}^{(i)}\)。随机效应服从零均值高斯分布，方差由矩阵 \(\mathbf{\Gamma}\) 控制。关键约束：固定效应图 \(G(\mathbf{B})\) 与随机效应图 \(G(\mathbf{\Gamma})\) 的**并集**必须是无环DAG。
- **关键技术细节**：
  - **模型形式**：基于似然的线混合结构方程模型，对每个节点 \(k\) 和集群 \(i\)，条件分布为高斯分布，协方差矩阵含随机效应项。
  - **可微无环约束**：采用**log-determinant acyclicity characterization**（Bello et al., 2022），定义 \(h(\mathbf{B},\mathbf{\Gamma}) = -\log\det(\mathbf{I} - \mathbf{B}\odot\mathbf{B} - \mathbf{\Gamma})\)，证明在定义域内 \(h=0\) 当且仅当并图无环。
  - **目标函数**：负对数似然 + L1稀疏惩罚 + 无环约束，即：
    \[
    \min_{\mathbf{B},\mathbf{\Gamma}\ge 0} \ell(\mathbf{B},\mathbf{\Gamma}) + \lambda_1\|\mathbf{B}\|_1 + \lambda_2\|\mathbf{\Gamma}\|_1 \quad \text{s.t. } h(\mathbf{B},\mathbf{\Gamma}) = 0.
    \]
- **算法流程**：
  1. **路径跟踪（Path-following）**：将无环约束作为障碍函数逐渐收紧，从 \(\mu=1\) 开始，每次乘0.1，共5步。
  2. **近端梯度下降**：将目标函数分解为光滑部分 \(g_\mu\) 和非光滑部分 \(r_\mu\)（L1惩罚和非负约束）。每次迭代执行梯度步后，对固定效应施加软阈值（soft thresholding），对随机效应施加非负软阈值。
  3. **批处理更新**：利用张量操作将集群数据对齐，使用批量 Cholesky 分解或 Woodbury/Sylvester 恒等式并行计算协方差矩阵的逆和行列式，显著降低计算复杂度。
  4. **后处理**：硬阈值（<0.05置零）和“抛光”（polishing）：在选定结构上去除惩罚重新优化，消除L1偏差。
- **理论保证**：给出可证明收敛的一阶方法（Proposition 2, Theorem 3），证明参数可识别性（Theorem 4），以及参数估计和结构恢复的渐近一致性（Theorem 5）。

## 6. 论文的主要结论与发现

- **Mixed DAG 显著优于 Fixed DAG**：在合成数据中，当样本量增大时，Fixed DAG无法恢复真实结构，而Mixed DAG能可靠学习，且估计误差、SHD、F1分数均大幅改善。
- **加入无环约束至关重要**：Mixed DG（无约束）表现最差，证明无环性对因果结构学习不可或缺。
- **接近理想上限**：Mixed DAG 的性能逼近使用真实拓扑顺序的 OM 方法，说明联合学习顺序与混合效应带来的统计损失很小。
