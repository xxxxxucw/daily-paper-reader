## 摘要
多任务回归旨在联合求解多个回归问题（称为任务）。与分别求解每个任务相比，只要任务之间充分相关，就能获得更好的性能。全共形预测是一种框架，它构建了一个数据依赖的预测区域，该区域以任意给定的置信水平包含未知的输出向量。然而，该预测区域的显式计算通常是难以处理的，因为它需要训练无限多个预测器。本文关注向量值函数的再生核希尔伯特空间中的多任务回归。通过设计一个包含全共形预测区域的近似预测区域来解决这一计算问题。该构建在两种情形下进行：(i) 当任务间协方差矩阵已知时；(ii) 当该矩阵被估计时。在体积方面，第一种情形下通过上界从理论上评估了该近似的紧致性。实验也证明，在两种情形下，该方法在合成数据上都优于分割共形预测。

## Abstract
Multi-task regression aims at jointly solving multiple regression problems, called tasks. Compared to solving each task separately, better performances can be achieved as long as the tasks are sufficiently related. Full-conformal prediction is a framework that formulates a data-dependent prediction-region containing the unknown output-vector at any prescribed confidence level. However, explicit computation of this prediction-region is intractable in general since it requires training infinitely many predictors. The present work focuses on multi-task regression in a Reproducing Kernel Hilbert Space (RKHS) of vector-valued functions. This computational issue is addressed by designing an approximating predictionregion containing the full-conformal one. This construction is carried out in two scenarios: piq when the inter-task covariance-matrix is known, and piiq when this matrix is estimated. In terms of volume, the tightness of this approximation is assessed theoretically by means of an upper-bound in the first scenario. It is also empirically proved to improve upon the split-conformal prediction on synthetic data in both scenarios.

## 2. 方法论
### 核心思想
- 利用**算法稳定性**（uniform stability）导出预测器对训练数据微小变化的敏感度上界，从而构造非一致性分数的**上下近似**，进而得到包含/被包含于真实FullCP区域的上下近似FullCP区域。
- 近似区域的计算只需训练一次（使用训练集D），无需对每个候选y重新训练。

### 关键技术细节
1. **模型设定**：向量值函数属于RKHS H，对应矩阵值核 K(x,t)，正则化经验风险最小化得到预测器 $\hat{f}_{\lambda;D}$。损失函数只需凸、Lipschitz连续、下有界。
2. **非一致性分数**：采用马氏距离 $S_{\lambda;D_y}^{\Gamma}(x,u) = \|\Gamma^{-1/2}(u - \hat{f}_{\lambda;D_y}(x))\|$，其中 $\Gamma$ 为任务间协方差矩阵。
3. **稳定性界**：引理18给出 $\|\hat{f}_{\lambda;D_y} - \hat{f}_{\lambda';D}\|_H \leq \frac{\rho_p \|K(X_{n+1},X_{n+1})\|_{op}^{1/2}}{2\lambda (n+1)}$。
4. **近似分数构造**：
   - 已知 $\Gamma$ 时：上近似 $S^{up} = S_{\lambda';D}(\cdot) + \tau_\lambda(\cdot)$，下近似 $S^{lo} = S_{\lambda';D}(\cdot) - \tau_\lambda(\cdot)$，其中 $\tau_\lambda(x)$ 为稳定性界导出的校正项。
   - 估计 $\Gamma$ 时：额外引入乘法校正和分母（依赖于测试输出y），利用Sherman-Woodbury-Morrison公式处理协方差矩阵更新。
5. **近似区域表达**：上下近似FullCP区域均为椭球体（或全局Y），其半径由校正后的经验分位数决定（见命题23, 32, 39, 40）。

### 算法流程（文字说明）
1. 输入：训练数据D，测试输入 $X_{n+1}$，置信水平 $\alpha$，正则化参数 $\lambda$。
2. 训练一个预测器 $\hat{f}_{\lambda';D}$（使用调整后的正则化 $\lambda'=\frac{n+1}{n}\lambda$）。
3. 计算每个训练点的非一致性分数 $S_{\lambda';D}(X_i,Y_i)$。
4. 根据分数排序确定分位数，并加上/减去稳定性校正项，得到上/下近似区域半径。
5. 输出一个中心为 $\hat{f}_{\lambda';D}(X_{n+1})$、形状由 $\Gamma$ 或 $\hat{\Gamma}_a$ 决定的椭球体作为上近似FullCP区域（保证覆盖）。
