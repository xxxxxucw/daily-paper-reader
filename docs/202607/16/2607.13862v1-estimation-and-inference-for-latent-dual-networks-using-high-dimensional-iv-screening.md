## 摘要
我们开发了一种新的方法论，用于具有潜在双重结构的高维面板网络模型的估计与推断。该框架允许结果同时受到正向和负向互动渠道的影响，适应了某些互动增强结果而其他互动产生竞争和置换效应的设置。所提出的方法直接从结构模型中使用观测数据识别和估计网络，无需预先指定网络。通过网络恢复通过顺序工具变量筛选过程实现。我们建立了精确的支持恢复和先知等价的后选择推断。应用于美国公司杠杆数据的应用揭示了公司财务决策中增强和置换互动的共存。

## Abstract
We develop a novel methodology for estimation and inference in high-dimensional panel network models with latent dual structures. The framework allows outcomes to be affected simultaneously by positive and negative interaction channels, accommodating settings in which some interactions reinforce outcomes while others generate competition and displacement effects. The proposed method identifies and estimates the network directly from the structural model using observed data without the need to pre-specify the network. Network recovery is achieved through a sequential instrumental-variable screening procedure. We establish exact support recovery and oracle-equivalent post-selection inference. An application to U.S. corporate leverage data reveals the coexistence of reinforcing and displacement interactions in firms' financial decisions.

## 1. 论文的核心问题与整体含义（研究动机和背景）
- **核心问题**：在面板数据网络模型中，观测结果可能同时受到**正向互动（增强效应）** 和**负向互动（置换效应）** 的影响，但传统方法要么假设网络结构已知，要么仅考虑单一符号的交互（如所有链接均产生同向影响）。如何从数据中直接识别并区分这两种潜在交互网络，是一个关键挑战。
- **研究动机**：现实经济场景（如企业财务决策、社会互动、空间经济）中，企业间可能通过模仿、学习产生协同（增强），也可能通过竞争、市场份额重新配置产生置换（竞争）。现有网络恢复方法（如Lasso型）常将负向估计值视为噪声或强制归为弱正向，无法刻画双重机制。
- **整体含义**：本文提出 **BOLMT（Boosting One-Link-at-a-Time with Multiple Testing）** 算法，利用工具变量筛选逐步恢复稀疏交互矩阵，并依据系数符号分解为增强网络和置换网络，实现了**无需预设网络结构、且能共存正负交互**的建模与推断。

## 2. 论文提出的方法论：核心思想、关键技术细节
### 2.1 核心思想
- 建模：  
  \[
  y_{i,t} = \eta_i + \beta_i' x_{i,t} + \rho_1 \sum_{j\neq i} w_{1,i,j} y_{j,t} + \rho_2 \sum_{j\neq i} w_{2,i,j} y_{j,t} + \varepsilon_{i,t}
  \]
  其中 \(\rho_1 > 0\)（增强），\(\rho_2 < 0\)（置换），且两种网络的支撑互不相交。
- 等价表述为：  
  \[
  y_{i,t} = \eta_i + \beta_i' x_{i,t} + \sum_{j\neq i} \alpha_{i,j} y_{j,t} + \varepsilon_{i,t}, \quad \alpha_{i,j} = \rho_1 w_{1,i,j} + \rho_2 w_{2,i,j}
  \]
  核心目标是恢复稀疏矩阵 \(A = (\alpha_{i,j})\)，然后按符号分解。

### 2.2 关键技术细节 – BOLMT算法
- **行式恢复**：对每个单元 \(i\)，将其余 \(N-1\) 个候选视为潜在链接，逐一步骤选择。
- **步骤**：
  1. 初始化已选集 \(H_{i,0} = \emptyset\)。
  2. 对剩余候选 \(j \notin H_{i,m-1}\)，计算**部分IV统计量** \(t_{i,j,H_{i,m-1}}\)（基于工具变量拟合值，并投影掉已选控制变量）。
  3. 选择统计量绝对值最大的候选 \(j^* = \arg\max |t_{i,j}|\)。
  4. 若 \(|t_{i,j^*}| > c_N\)（多重检验阈值，\(\propto \sqrt{\log N}\)），则将 \(j^*\) 加入已选集，返回步骤2；否则停止，输出当前已选集作为 \(\hat{S}_i\)。
- **后选择分解**：对每个选出的链接，估计系数 \(\hat{\alpha}_{i,j}\)，按符号分为正（增强网络）和负（置换网络）。行内归一化得到 \(\hat{\rho}_{1,i}, \hat{\rho}_{2,i}\) 及权重。
- **均值组（MG）聚合**：将各单元估计进行简单平均。

### 2.3 理论保证
- 在稀疏性、检测性、代理对齐等假设下，定理3.1证明**精确支撑恢复**（\(\Pr(\hat{S}_i = S_i^n) \to 1\)）。
- 定理3.2证明符号支撑恢复及权重一致性。
- 定理3.3-3.4证明**oracle等价性**：后选择IV估计量与已知真网络下的oracle估计渐近相同，均值组估计量渐近正态。
