## 摘要
实证研究中通常只观察到被选择单位的结果，而处理可能会改变哪些单位被观察到。本文研究在单侧选择下的随机化研究中的预测问题。标准的预测区间可能失败，因为处理组中被选择的观测与对照组中被选择的观测不是同一组。本文探讨如何预测始终被观测单位的缺失处理结果和个体处理效应。提出的基于共形化的李氏程序利用处理组中被选择的观测来训练和检验任意预测规则，然后通过观测到的处理-对照组选择差距调整阈值。对于被选择的对照组，缺失处理结果的区间通过观测到的未处理结果进行平移，从而产生个体处理效应区间。该方法在不要求预测规则正确指定的情况下提供可靠的覆盖。关键结果表明，所提出的调整使用了李氏[2009]单调选择逻辑所隐含的精确不确定性量。在模拟中，普通的共形预测在选择引起的分布偏移下表现出较低的覆盖率，而李氏调整方法达到了期望的覆盖率。结果表明，所提出的选择校正方法可以支持可靠的反事实预测，同时保留与现代预测工具的实际实现。

## Abstract
Empirical studies often observe outcomes only for selected units, and treatment may change who is observed. This paper studies prediction in randomized studies with one-sided selection. Standard prediction intervals can fail because treated selected observations are not the same group as selected controls. The paper asks how to predict missing treated outcomes and individual treatment effects for always-observed units. The proposed conformalized Lee procedure uses treated selected observations to train and check any prediction rule, then adjusts the cutoff using the observed treatment-control selection gap. For selected controls, the missing treated-outcome interval is shifted by the observed untreated outcome to produce an individual treatment-effect interval. The method provides reliable coverage without requiring the prediction rule to be correctly specified. The key result shows that the proposed adjustment uses the exact amount of uncertainty implied by the monotone selection logic of Lee [2009]. In simulations, ordinary conformal prediction demonstrates a lower coverage rate under selection-induced distribution shift, while the Lee-adjusted methods achieve the desired coverage rate. The results show that the proposed selection correction method can support reliable counterfactual prediction, while retaining practical implementation with modern prediction tools.

## 1. 核心问题与整体含义（研究动机和背景）
在随机化实验中，样本选择是一个普遍问题：只有被“选中”的单位才能观测到结果变量，而处理本身可能改变谁被选中。例如，某一政策干预可能促使原本不汇报结果的单位开始汇报，导致处理组中被观察到的样本与对照组中被观察到的样本来自不同的潜在人群。标准预测区间（如分裂共形预测）假定校准数据与目标数据同分布，但在这种“单调选择”下，该假设不成立，因此预测区间失效。

论文的核心目标是：**为“始终被选择”（always-selected）的单位，构造无分布假设（distribution-free）的预测区间**，以估计其缺失的处理组潜在结果以及个体处理效应（Individual Treatment Effect, ITE）。这一问题属于部分识别（partial identification）范畴，因为处理组中被选择的观测是始终被选择群体和边际诱导群体（marginal-in）的混合，目标分布无法点识别。

整体含义：本文通过结合 Lee（2009）的单调选择模型与共形预测框架，发展了一套理论严谨、不依赖模型正确设定、且能提供有限样本覆盖保证的反事实预测方法，为处理样本选择偏差下的个体级推断提供了新工具。

## 2. 方法论：核心思想、关键技术细节
### 核心思想
- **单调选择假设**（Assumption 2）：`S(1) ≥ S(0)` 几乎必然，即处理只会增加被选择的概率，不会减少。因此，对照组中被选择的单位全部属于“始终被选择”层（stratum A）；处理组中被选择的单位则是“始终被选择”（A）与“边际诱导”（M）的混合。
- **模糊集刻画**：设处理组被选择分布为 `P = L(X, Y(1) | S(1)=1)`，目标分布（始终被选择层）为 `Q = L(X, Y(1) | A)`。由单调性可得 `π = p0/p1`（p0 = Pr(S(0)=1)，p1 = Pr(S(1)=1)）是处理组被选择样本中始终被选择的比例。进一步得到似然比约束：
  ```
  Q ≪ P, 0 ≤ dQ/dP ≤ 1/π  P-几乎必然
  ```
  即 Q 属于 **Lee 模糊集** `Q(P,π)`。命题1和2证明该集合是**锐化（sharp）的**——所有满足该约束的分布均可由某个满足假设的数据生成过程实现。

### 关键技术细节：共形化 Lee 过程（Conformalized Lee Procedure）
论文采用 **分裂共形预测** 框架，主要调整在于**校准阈值**的选择。

1. **分裂样本**：将处理组被选择观测（I1）随机分为训练集（Itrain）和校准集（Ical）。
2. **训练预测规则**：基于 Itrain 拟合任意模型（如回归），定义非一致性得分函数 V(x, y)（如绝对残差）。
3. **校准**：在 Ical 上计算得分 `V_i`，升序排列得 `V_(1) ≤ ... ≤ V_(m)`，并令 `V_(m+1)=+∞`。
4. **Lee 调整的阈值**：设定索引 `k = ⌈(m+1)(1 - απ̂)⌉`，其中 `π̂` 是 π 的估计值。阈值 `t̂ = V_(k)`。
5. **输出预测集**：对于新单位（来自始终被选择层）的协变量 x，预测集为 `Ĉ₁(x) = {y : V(x,y) ≤ t̂}`。

当 `π̂ = 1` 时退化为普通共形预测；当 `π̂ < 1` 时阈值更保守（等价于用 `(1-απ̂)` 分位数代替 `(1-α)` 分位数）。

**覆盖保证**：定理1证明，若 `π̂ ≤ π`，则条件覆盖概率至少为 `1-α`；若 `π̂ > π`，则覆盖误差有界。进一步利用 **Clopper-Pearson 或 Hoeffding 下界**构造 `π̂` 的保守估计，可以获得高概率覆盖。

**个体处理效应区间**：对于对照组被选择单位（已知Y(0)），直接定义 `Ĉτ(x, y0) = Ĉ₁(x) - y0`，由推论1得到边际ITE区间保证。

**最优性**：命题3和4证明，`(1-απ)` 分位数是**最小最大最优的**——任何更小的阈值都会在某些模糊集内分布下导致覆盖不足。

## 3. 实验设计
- **模拟设置**：采用蒙特卡洛模拟，重复次数 R=100 次。
- **数据生成（DGP）**：
  - 协变量 X ~ Unif([0,1]⁴)
  - 潜在结果 Y(1) = βᵀX + U₁，其中 U₁|X ~ N(0, σ²(X))，σ²(X) = 1 + 12(2.5X₁)²；Y(0) 含额外噪声。
  - 处理组选择率 p1=0.8，处理分配概率 0.5。
  - 四种 DGP 区分始终被选择层 A 的分配规则：
    1. **Benign（良性）**：A ~ Bernoulli(π) 独立于 (X, U₁, Y(1))，分布无偏移。
    2. **Conditional tail（条件尾）**：A 指示 |U₁| > 条件高分位数，导致得分尾部集中。
    3. **Unconditional tail（无条件尾）**：A 指示 |U₁| > 无条件高分位数，同时引起协变量偏移（因 σ²依赖于 X₁）。
    4. **Smooth（平滑）**：A 概率由 X₁ 和标准化残差的 Logistic 函数决定，模拟平滑异质选择。
- **评估指标**：经验覆盖率与目标覆盖率之差（Gap）、区间长度、无限区间概率。
- **对比方法**：
  - **Naive conformal**：使用 `π̂=1`（即忽视选择偏差）。
  - **Oracle Lee**：使用真实 π（已知）。
  - **Plug-in Lee**：使用 π̂ = min{1, p̂₀/p̂₁}，其中 p̂ₐ 为经验选择率。
  - **CP-Lee**：使用 Clopper-Pearson 下界 `π̂CP`，目标覆盖率调整为 `1-α-δπ`（δπ=0.05）。
  - **Hoeffding-Lee**：使用 Hoeffding 下界 `π̂H`。
- **非一致性得分**：Oracle 残差（真实 β）、拟合残差（估计 μ̂₁）、CQR（共形分位数回归）。
- **参数网格**：π ∈ {0.25, 0.50, 0.75}，校准样本量 m ∈ {100, 200}，目标覆盖率 1-α ∈ {0.5,0.6,0.7,0.8,0.9}。
