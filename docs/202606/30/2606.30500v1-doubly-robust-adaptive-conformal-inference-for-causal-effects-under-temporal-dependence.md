## 摘要
我们提出双重稳健自适应共形推断（DR-ACI），该方法在时间依赖性下为双重稳健伪结果构建预测区间。

## Abstract
We propose doubly robust adaptive conformal inference (DR-ACI), which constructs prediction intervals for doubly robust pseudo-outcomes under temporal dependence.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：在时间依赖数据（如金融、流行病学数据）中量化因果效应（如条件平均处理效应 CATE）的不确定性时，现有方法面临两大挑战：一是数据非独立性（违反 exchangeability），二是模型误设定。传统的置信区间（如渐近、bootstrap、贝叶斯）依赖于正确模型设定或渐近理论，而机器学习估计量在时间依赖下难以提供分布自由的有限样本保证。
- **整体含义**：本文旨在为时间序列因果推断提供一种**分布自由、有限样本有效**的预测区间方法，能够处理数据的时间依赖性和模型误设定，同时保持对潜在因果效应的渐近覆盖解释。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- 结合**双重稳健（Doubly Robust）伪结果**（ψ_DR）与**自适应共形推断（Adaptive Conformal Inference, ACI）**，为伪结果构建在线更新的预测区间，并利用**时间块交叉拟合（temporal block cross-fitting）** 与**保护带（guard bands）** 处理时间依赖性。
- 理论保证：将覆盖缺口分解为三个可量化项：混合缺口（mixing gap）、干扰偏差税（nuisance-bias tax）和自适应率（adaptation rate）。

### 关键技术细节
1. **双重稳健伪结果**：
   - 定义：$\psi^{DR}_t = \hat{\mu}_1(X_t) - \hat{\mu}_0(X_t) + \frac{W_t(Y_t - \hat{\mu}_1(X_t))}{\hat{e}(X_t)} - \frac{(1-W_t)(Y_t - \hat{\mu}_0(X_t))}{1-\hat{e}(X_t)}$
   - 性质：若任一个干扰模型（倾向得分或条件均值）正确，则偏差为乘积形式 $\|\hat{e} - e\| \cdot \|\hat{\mu} - \mu\|$，而非和形式。

2. **时间块交叉拟合与保护带**：
   - 将时间序列划分为 $K$ 个连续块，对每个块 $B_k$ 将其相邻 $g$ 个观测作为保护带（排除出训练集），剩余数据训练干扰模型。这保证训练与校准点的时间间隔至少为 $g$，利用 $\beta$-mixing 性质控制剩余依赖 $\beta(g)$。

3. **自适应共形推断（ACI）**：
   - 在线更新误覆盖率 $\alpha_t$: $\alpha_{t+1} = \alpha_t + \gamma(\alpha - \mathbf{1}\{\psi^{DR}_t \notin \hat{C}_t\})$，提供长期平均覆盖保证（可达 $1-\alpha$ 的确定性上界）。

4. **方差标准化（VS-DR-ACI）**：
   - 使用标准化得分 $s^{VS}_t = |\psi^{DR}_t - \hat{\tau}(X_t)| / \hat{\sigma}_\xi(X_t)$，其中 $\hat{\sigma}_\xi^2(x) = \frac{\text{Var}(Y|X=x,W=1)}{e(x)} + \frac{\text{Var}(Y|X=x,W=0)}{1-e(x)}$，使区间宽度自适应于局部噪声水平，避免 ACI 在分布漂移下发散。

### 算法流程（文字说明）
1. 输入：数据 $(X_t, W_t, Y_t)$，块数 $K$，保护带大小 $g$，误覆盖率 $\alpha$，学习率 $\gamma$，CATE 估计器 $\hat{\tau}$。
2. **阶段1**：时间块交叉拟合：将序列分为 $K$ 块，每块 $b$ 个观测。对每个块 $B_k$，定义保护带 $G_k$（前后各 $g$ 个点），在补集数据上训练干扰模型 $(\hat{e}_k, \hat{\mu}_{0,k}, \hat{\mu}_{1,k})$。
3. **阶段2**：构建双重稳健伪结果：对每个 $t$，使用对应块训练的模型计算 $\psi^{DR}_t$。
4. **阶段3**：在线共形校准：初始 $\alpha_1 = \alpha$。对每个 $t$，计算得分 $s_t = |\psi^{DR}_t - \hat{\tau}(X_t)|$（或标准化版本），设 $\hat{q}_t = \text{Quantile}_{1-\alpha_t}(s_1,\dots,s_{t-1})$，构建区间 $\hat{C}_t = [\hat{\tau}(X_t) - \hat{q}_t, \hat{\tau}(X_t) + \hat{q}_t]$，更新 $\alpha_{t+1}$。

## 3. 实验设计

### 数据集 / 场景
- **仿真实验**：基于德国联邦债券期货（RX1）高频数据校准的 DGP，包含 AR(1) 协变量（$\rho$ 从 0 到 0.99），GARCH(1,1) 误差，Student-t 分布（$\nu = 2.5$），以及非线性 CATE $\tau(x) = \sin(2\pi x_1) + 0.5 x_2$。样本量 $T=2000$，重复 500 次。
- **四种情景**：
  - A/B：平稳 AR(1) 依赖（$\rho=0$ 或 $\rho=0.95$）。
  - C：协变量漂移（iid 协变量，$t=T/2$ 后均值平移 $\delta=1$，违反平稳性假设）。
  - D：依赖+漂移（AR(1) $\rho=0.95$，$t=T/2$ 后线性漂移，违反平稳性假设）。
- **真实应用**：纳斯达克动态 M-ELO 滚动推出（2024年4-5月），使用 SEC MIDAS 数据（约 9200 只股票，每日面板）和 SEC Rule 605 数据（805 只股票，月度面板）。处理变量：订单类型（固定 vs 动态定时器）。结果指标：隐藏订单份额、零碎订单份额、交易所交易量等。

### 基准方法（Benchmark）
- **DR-ACI**（本文无方差标准化）
- **VS-DR-ACI**（本文方差标准化版本）
- **ACI (non-DR)**：自适应共形推断用于原始残差 $|Y_t - \hat{f}(X_t)|$
- **Split conformal**：前60%训练，后40%校准
- **Block bootstrap**：圆形块 bootstrap（块大小 $b = \lfloor T^{1/3}\rfloor$，$B=199$）
- **DML-Wald-PO**：基于 DML 的 Wald 置信区间，使用局部残差方差

所有方法均使用相同的干扰模型（梯度提升树 XGBoost）和 CATE 估计器（随机森林）。

## 6. 主要结论与发现

1. **理论贡献**：提出了覆盖缺口的三项分解（定理2），包含混合缺口 $\min_\tau \{\tau/T + 2\beta(\tau)\}$、干扰偏差税 $\|\hat{e}-e\|_2\|\hat{\mu}-\mu\|_2 + O(\beta(g)^{\delta/(2+\delta)})$ 和自适应率 $O(T^{-1/2})$。这是首个在 $\beta$-mixing 下对 DR 伪结果有覆盖保证的方法。

2. **仿真结果**：
   - 在平稳条件下，所有共形方法均达到有效覆盖；VS-DR-ACI 相比 Split conformal 区间宽度减少 63%（3.4 vs 9.1），且在强依赖（$\rho=0.99$）时仍保持 90% 覆盖（理论下界为 67%）。
   - 在依赖+漂移（情景 D）下，VS-DR-ACI 是唯一同时维持有效覆盖（89.9%）和稳定宽度（中位数 4.9）的方法；DR-ACI 宽度膨胀至 170.3，Split conformal 覆盖损失 29.4pp，DML-Wald-PO 损失 35.1pp。
   - 干扰模型误设定不影响伪结果覆盖（自适应分位数自动调整），但宽度增加约 7%。

3. **应用结果**：
   - 动态 M-ELO 使隐藏订单份额平均提高 3.6 个百分点，零碎订单份额降低 2.3 个百分点，交易所交易量降低 3.6%。
   - DR-ACI 区间宽度约为渐近区间的一半，33.2% 的股票 CATE 显著非零（基线仅 12.3%），对应约 2000 只额外可区分股票。

## 7. 优点

- **理论严谨**：首次在时间依赖因果推断中给出分布自由的有限样本覆盖保证，且界中各项物理意义明确。
- **方法实用**：VS-DR-ACI 在强依赖和漂移下仍保持稳健，解决了 ACI 在分布漂移下的发散问题（通过方差标准化保证得分有界）。
- **实验设计全面**：覆盖平稳、非平稳、弱/强依赖、漂移等多种场景，对比方法包括标准共形、bootstrap 和渐近方法。仿真 DGP 基于真实数据校准，接近实际。
- **应用有价值**：用于市场微观结构改革，估计股票级因果效应，并量化不确定性，识别出更多显著效果，具有操作意义。
- **代码开源**：提供 GitHub 仓库，便于复现。

## 8. 不足与局限

- **严格要求平稳性**：定理2 在定理层面要求严格平稳和 $\beta$-mixing，排除结构突变和长记忆过程。针对非平稳的 VS-DR-ACI 结果仅为实证，无理论保证。
- **理论绑定保守**：混合缺口绑定在实践中有很大松弛（理论预测覆盖可低至 67%，实际观察 90%），作者仅给出推测（Conjecture 4）但未证明。
- **异常性假设（SUTVA）**：在金融应用中，干扰（spillover）是重要问题，本文仅通过样本限制缓解，未提供正式识别或校正。
- **CATE 覆盖非直接**：区间校准的是伪结果而非真实 CATE，CATE 覆盖依赖于 $\hat{\tau}$ 的一致性（推论6），属于间接保证，不是有限样本的 $1-\alpha$ 覆盖。
- **算力未报告**：无 GPU 使用信息，难以判断大规模扩展成本。
- **真实应用对比有限**：仅与渐近区间对比，未与其它因果共形方法（如 Lei & Candès 2021）对比，因为那些方法假设交换性，在此不适用。但可以讨论。
